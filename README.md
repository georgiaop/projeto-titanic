# Projeto Titanic 🚢📊 
## Sobre o projeto

Projeto de análise de dados utilizando a base de passageiros do Titanic.

Durante o projeto foram aplicados conhecimentos de:
- tratamento de dados;
- análise exploratória;
- visualização gráfica;
- interpretação de dados utilizando Python.
  
--- 

## Ferramentas utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab
